# Sales Data Analysis using Excel & Power BI

This project analyzes sales data and presents insights through an interactive Power BI dashboard.

Tools Used:
- Microsoft Excel (Data Cleaning & Preparation + Pivot table)
- Power BI (Data Visualization & Dashboard)

Key Insights:
- Total Sales & Profit
- Region-wise Performance
- Top Selling Products
- Monthly Sales Trend

Dataset Size: 100+ rows
